# BhashaBridge

BhashaBridge is an AI-powered language translation system that understands meaning, corrects grammar, and translates text naturally between languages using the Grok API (xAI).

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Create a `.env` file from the template and add your API key:
   ```env
   GROK_API_KEY=your_actual_key_here
   ```

3. Run the translation script:
   ```bash
   python translator.py
   ```
